## 1. 产品概述

Python MCP Server 是一个基于 Docker 的容器化 Python 执行环境管理系统，旨在提供安全、隔离、可控制的远程 Python 代码执行能力。

### 核心价值
- **安全隔离**: 每个代码执行都在独立的容器环境中进行
- **资源控制**: 精确控制 CPU、内存、GPU 等资源使用
- **便捷管理**: 提供 Web 界面和 API 接口进行容器管理
- **预装依赖**: 内置丰富的数据科学和机器学习库

## 2. 核心功能

### 2.1 六大核心工具

#### Python代码执行
- 通过 HTTP API 远程执行 Python 代码
- 支持同步和异步执行模式
- 实时输出执行结果

#### 终端命令执行
- 执行任意 Linux 终端命令
- 支持复杂的 shell 脚本运行
- 实时返回命令执行结果

#### 文件管理
- 列出容器内指定目录的文件列表
- 支持文件下载功能
- 提供文件浏览和传输能力

#### 依赖包管理
- 动态安装 Python 包
- 预装常用数据科学库（pandas, numpy, torch, tensorflow等）
- 支持私有包仓库

#### 健康检查
- 实时监控容器运行状态
- 提供系统资源使用情况
- 支持自动故障检测和恢复

#### 资源限制与隔离
- **CPU 核心数限制**: 可设定容器使用的 CPU 核心数量
- **内存限制**: 可设定容器最大内存使用量
- **GPU 显存限制**: 支持 NVIDIA GPU 显存分配
- **存储空间限制**: 可设定容器存储空间上限
- **运行时间限制**: 自动超时停止，防止长时间占用

## 3. 技术架构

### 3.1 整体架构
```
用户请求 → Nginx代理 → MCP容器集群 → Docker引擎
              ↓
        容器管理界面(Gradio)
```

### 3.2 核心组件

#### 3.2.1 容器管理前端 (container_manager.py)
- 基于 Gradio 构建的 Web 界面
- 提供容器创建、监控、管理功能
- 实时显示容器状态和资源使用情况

#### 3.2.2 MCP 服务器 (mcp_server.py)
- 基于 Flask 的 RESTful API 服务
- 提供代码执行、包安装等核心功能
- 支持 WebSocket 和 SSE 流式传输

#### 3.2.3 Nginx 代理层
- 动态代理配置生成
- 负载均衡和请求转发
- 支持多容器实例统一接入

#### 3.2.4 Docker 容器引擎
- 容器生命周期管理
- 资源限制和隔离
- 自动扩缩容支持

### 3.3 部署架构
```
┌─────────────────┐    ┌──────────────────┐    ┌────────────────────┐
│   用户客户端    │───▶│   Nginx反向代理  │───▶│  MCP容器管理集群   │
└─────────────────┘    └──────────────────┘    └────────────────────┘
                            │                         │
                            ▼                         ▼
                    ┌──────────────────┐    ┌────────────────────┐
                    │  容器管理界面    │    │   Docker容器池     │
                    └──────────────────┘    └────────────────────┘
```

- 数值计算和仿真实验
- 并行计算任务分发
- 科学研究协作平台

### 4.6 MCP集成自动化
通过MCP与Agent自动调用工具，可以完成复杂业务流程自动化：

#### 数据分析全流程
- **数据预测**: 基于历史数据进行趋势预测和模型训练
- **数据探查**: 自动识别数据特征、异常值和分布情况
- **数据分析**: 执行统计分析、相关性分析等高级分析任务
- **可视化**: 自动生成图表、仪表板和交互式可视化界面
- **报告生成**: 输出Excel/PPT格式的专业分析报告

#### 应用开发
- **前端界面制作**: 结合Web框架快速构建交互式前端应用
- **小游戏开发**: 利用Pygame等库开发简单的娱乐游戏
- **自动化脚本**: 批量处理文件、数据转换等重复性工作

#### 系统集成
- **跨平台操作**: 无缝集成Linux、Windows等不同操作系统环境
- **第三方服务对接**: 自动调用API、数据库查询等外部服务
- **DevOps自动化**: CI/CD流程中的代码测试和部署

几乎所有与Python相关的任务都可以通过MCP自动化完成，大大提升工作效率。

## 5. 产品亮点

### 5.1 安全可靠
- **容器隔离**: 每次执行都在独立的安全沙箱中进行
- **资源限制**: 防止恶意代码耗尽系统资源
- **自动清理**: 超时自动停止，避免僵尸进程

### 5.2 高效便捷
- **一键部署**: 简单的启动脚本，快速搭建服务
- **丰富预装**: 内置 20+ 常用数据科学库，开箱即用
- **多语言支持**: 支持 Python 及其生态系统

### 5.3 灵活扩展
- **参数化配置**: 支持灵活的资源配置选项
- **插件化设计**: 易于扩展新功能和服务
- **API 驱动**: 完整的 RESTful API 接口

### 5.4 智能管理
- **可视化界面**: 直观的 Web 管理界面
- **实时监控**: 容器状态和资源使用实时查看
- **自动化运维**: 自动生成 Nginx 配置，动态更新代理规则

### 5.5 性能优越
- **并发支持**: 支持多容器并发执行
- **GPU 加速**: 充分利用硬件加速能力
- **低延迟响应**: 优化的网络传输和执行效率

### 5.6 自动化革命
- **全流程自动化**: 彻底解决Python代码不能直接执行的问题，实现从代码编写到执行的全自动化流程
- **智能环境配置**: 自动安装所需的Shell环境和依赖包，无需手动配置复杂的运行环境
- **代码智能流转**: 消除手动复制粘贴代码的繁琐步骤，实现代码的自动传递和执行
- **无人值守操作**: 支持定时任务和条件触发，实现真正的无人值守自动化执行
